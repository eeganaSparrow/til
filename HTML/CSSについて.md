# CSSについて（覚え書き）
1. 文字
    * font-size（文字サイズ）
        * px・・・絶対値
        * em・・・親要素に対する相対値
        * rem・・・html要素に対する相対値で、ブラウザ設定のサイズに対応
        * いろんな環境に対応するためには相対値による指定がいいのかも。pxの使い所は考える必要がある。
    * font-family（フォント）
    * Webフォントはlinkに指定
    * font-weight（太さ）
    * line-height（行高さ）
    * text-align（行の整列）
2. 背景
    * backgroud
    * 色やサイズ、位置などを設定
3. 余白
    * margin・・・要素外の余白
    * padding・・・要素内の余白
4. 線
    * border・・・線種や太さなどを設定
5. リスト
    * list-style-type・・・リストの種類を設定
